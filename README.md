## 이 레포에 대해

1인 개발자가 **AI 오케스트레이션**을 활용해 게임을 만드는 전 과정을 투명하게 공개합니다.
기획 의도, 설계 결정, 실패와 회고까지 — 포트폴리오이자 개발 로그입니다.
