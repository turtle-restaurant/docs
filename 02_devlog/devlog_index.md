## 작성 규칙

- 파일명: `YYYY-MM-DD.md`
- 같은 날 추가 작업은 해당 날짜 파일에 누적 업데이트
- 하루 작업 마무리 시 Claude에게 요청:

```
오늘 작업 일지 작성하고 푸시해줘.
한 일: [작업 내용]
결정 사항: [결정한 것들]
내일 할 일: [다음 작업]
```
