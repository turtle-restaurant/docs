## 1. 전체 비주얼 방향

### 1.1 키워드

```
따뜻함 / 손그림 / 수채화 / 먹음직스러운 / 아기자기 / 일러스트적 / 정성스러운
```

### 1.2 레퍼런스

- **음식 레퍼런스**: 첨부 고등어구이 이미지
  - 수채화 + 손그림 질감 혼합
  - 선화 없이 색의 번짐과 붓터치로 외곽 표현
  - 실사에 가까운 색감이지만 단순화된 일러스트 느낌
  - 접시/그릇까지 포함해 하나의 완성된 음식 일러스트로 표현
- **캐릭터/배경 레퍼런스**: 아기자기한 구도의 귀여운 동물 캐릭터, 따뜻한 분위기
- **전체 방향**: 음식은 수채화 손그림 일러스트, 캐릭터는 단순한 만화풍, 배경은 따뜻하고 정돈된 느낌

### 1.3 한 줄 방향

> "수채화로 그린 듯 정성스럽고 먹음직스러운 음식이 있는 아기자기한 식당"

### 1.4 스타일 특징 — 카테고리별 구분

| 카테고리 | 스타일 |
|----------|--------|
| 음식 | 수채화 손그림 일러스트, 선화 없거나 흐릿하게, 붓터치 질감, 실사적 색감 |
| 캐릭터 | 굵은 선화, 플랫 셀 셰이딩, 귀엽고 단순한 만화풍 |
| 배경/UI | 따뜻한 플랫 + 얇은 선화, 채도 낮게 유지 |

## 3. 음식 이미지 스타일 (핵심)

> 레퍼런스: 첨부 고등어구이 이미지

### 3.1 렌더링 방식

- **기법**: 수채화 손그림 일러스트
- **선화**: 없거나 매우 흐릿하게 — 외곽은 색의 번짐으로 자연스럽게 처리
- **음영**: 수채화 번짐 + 붓터치 흔적으로 입체감 표현 (하드한 셀 셰이딩 금지)
- **하이라이트**: 흰 붓터치 또는 흰 여백으로 광택 표현
- **질감**: 붓터치 흔적이 보이는 손그림 느낌 유지
- **배경**: 흰색 또는 투명 (접시/그릇 포함한 완성 일러스트로 표현)

### 3.2 접시/그릇 형태 (GDD_game 메뉴 구성 기준)

GDD의 메뉴 카테고리별 접시 형태를 반드시 준수.

| 카테고리 | 접시 형태 | 비고 |
|----------|----------|------|
| 밥류 | 동그라미 공기 | 뚜껑 없음, 돔형 밥 |
| 메인반찬 (불고기/생선구이/닭갈비) | 높이가 긴 직사각형 접시 | 레퍼런스 고등어구이 이미지 기준 |
| 국/찌개류 | 동그라미 그릇/뚝배기 | 국물이 보이도록 위에서 살짝 내려다본 앵글 |
| 서브반찬 | 1×2 분할 직사각형 접시 | 좌우 2칸으로 나뉜 작은 직사각형 |

### 3.3 음식별 상세 스타일 메모

| 음식 | 접시/그릇 | 렌더링 포인트 |
|------|----------|------------|
| 쌀밥 | 흰 공기, 동그라미 | 돔형 밥, 흰 하이라이트, 따뜻한 밥알 질감 |
| 잡곡밥 | 흰 공기, 동그라미 | 아이보리 베이스 + 브라운/블랙 잡곡 점 |
| 미역국 | 흰 사발, 동그라미 | 진한 올리브그린 국물, 소고기 조각, 파 |
| 김치찌개 | 뚝배기, 동그라미 | 레드오렌지 국물, 김치, 두부, 파, 테두리 그을린 뚝배기 |
| 콩나물국 | 흰 사발, 동그라미 | 맑은 연노랑 국물, 흰 콩나물, 빨간 고추 점 |
| 불고기 | 직사각형 접시 (높이긴) | 글레이즈 다크브라운, 파/양파, 고기 결 |
| 생선구이 | 직사각형 접시 (높이긴) | 골든 브라운, 구운 자국, 고등어구이 이미지 스타일 직접 참고 |
| 닭갈비 | 직사각형 접시 (높이긴) | 오렌지레드 양념, 깻잎, 쫄깃한 고기 표면 |
| 배추김치 | 1×2 분할 직사각형 | 레드코랄, 배추 흰 줄기, 양념 번짐 |
| 쌈채소 | 1×2 분할 직사각형 | 진한 초록, 상추 결, 여러 잎 겹친 표현 |

## 4. 캐릭터 스타일

### 4.1 전체 방향

- 동물의 숲 주민처럼 통통하고 단순한 형태
- 외곽선: 굵고 또렷 (다크 브라운 `#4A2E1A`, 2~3px)
- 음영: 셀 셰이딩 2단계 (기본색 + 그림자색 1단계)
- 눈: 크고 동그랗게, 하이라이트 1개
- 코/입: 최소화
- 볼터치: 필수 (소프트 핑크 `#E8A0A0`, 반투명 원)
- 비율: 머리 : 몸 = 1 : 1 ~ 1 : 1.2 (SD 비율)

### 4.2 일반 손님 4종

| 캐릭터 | 주조색 | 특징 |
|--------|--------|------|
| 참새 | 브라운 + 아이보리 | 노란 부리, 작은 날개 |
| 쥐 | 미디엄 그레이 | 동그란 귀, 긴 꼬리 |
| 고양이 | 크림/오렌지/그레이 | 뾰족한 귀, 수염 |
| 강아지 | 베이지/브라운 | 처진 귀, 촉촉한 눈 |

### 4.3 에피소드 손님

- 일반 손님보다 약간 크고 디테일 추가
- 개성 있는 소품 포함 (구상 중)
- 표정 변화 있음 (기본/기쁨/슬픔/놀람)

## 5. 배경/환경 스타일

### 5.1 식당 내부

- 원근감: 살짝 위에서 내려다보는 2.5D 뷰
- 외곽선: 얇은 선화 적용 (음식보다 덜 강조)
- 벽/바닥: 따뜻한 나무 질감, 크림/베이지 계열
- 조명: 노란빛 따뜻한 포인트 조명 표현
- 소품: 작은 화분, 액자, 간판 등 아기자기한 요소
- 채도: 캐릭터/음식보다 낮게 유지

### 5.2 메인 게임 화면

- 조리기구 영역: 주방 느낌의 따뜻한 브라운 계열
- 주문 리스트 영역: 메모지/화이트보드 질감, 굵은 테두리
- 전체 배경: 게임 플레이 요소가 잘 보이도록 채도 낮게 유지

## 6. UI 스타일

### 6.1 버튼

- 형태: 모서리가 둥근 라운드 렉탱글
- 외곽선: 다크 브라운 (`#4A2E1A`), 두께 2~3px
- 배경: 크림 또는 피치 계열 단색 + 하단 두꺼운 테두리로 입체감
- 눌림 효과: 하단 테두리 사라지며 아래로 2px 이동

### 6.2 아이콘

- 선화 기반 단순 일러스트, 굵은 외곽선
- 내부 플랫 + 셀 셰이딩 1단계
- 크기 통일: 48×48px 기준

### 6.3 폰트 방향

- 손글씨 느낌의 둥근 볼드 폰트
- 국문: 둥근 고딕 볼드 계열 (예: Nanum Round ExtraBold, 온글잎)
- 영문: 둥글고 두꺼운 손글씨 계열
- 크기 계층: 타이틀 32px / 본문 20px / 보조 16px (1920×1080 기준)

### 6.4 주문서 UI

- 메모지/영수증 질감의 아이보리 배경, 굵은 테두리
- 핀으로 꽂힌 듯한 연출
- 특별 손님 주문서: 연보라 (`#B0A0D8`) 배경으로 구분

## 7. 애니메이션 방향

| 요소 | 방향 |
|------|------|
| 캐릭터 대기 | 살랑살랑 좌우 흔들림, 0.8~1.0초 주기 |
| 조리 중 (냄비) | 뚜껑이 통통 튀는 애니메이션, 보글보글 이펙트 |
| 조리 중 (프라이팬) | 재료가 지글지글 흔들림, 연기 이펙트 |
| 조리 중 (밥솥) | 수증기 모락모락 |
| 조리 완성 | 반짝이는 별 이펙트 1~2개 |
| 탄 음식 | 검은 연기 이펙트 |
| 서빙 완료 | 하트 또는 별 1개 팝업 |
| 손님 퇴장 (화남) | 화남 이모티콘 팝업 + 빠른 퇴장 |
| 버튼 탭 | 살짝 눌리는 스케일 애니메이션 (0.95배) |
| 이징 | ease-out 기반, 통통 튀는 느낌은 bounce |
