## 1. 전체 비주얼 방향

### 1.1 키워드

```
따뜻함 / 귀여움 / 아기자기 / 만화풍 / 굵은 선화 / 사실적이지만 단순한 / 먹음직스러운
```

### 1.2 레퍼런스

- 음식 레퍼런스: 첨부 미역국 이미지 — 굵은 외곽선, 단순화된 음영, 채도 있는 색감
- 캐릭터/배경 레퍼런스: 토끼 요리 게임 이미지 — 아기자기한 구도, 따뜻한 분위기
- 전체 방향: 캐릭터와 배경은 귀엽고 아기자기하게, 음식은 먹음직스럽고 또렷하게

### 1.3 한 줄 방향

> "귀엽고 아기자기한 식당에서 진짜 맛있어 보이는 음식을 파는 만화풍 게임"

### 1.4 스타일 특징

- 외곽선: 굵고 또렷한 검정/다크브라운 선화 (캐릭터·음식·UI 모두)
- 음영: 플랫 셀 셰이딩 (2~3단계, 그라데이션 없음)
- 채도: 전체적으로 중간 채도 유지, 파스텔보다 진하고 실사보다 단순하게
- 음식: 실제 색감에 가깝게 채도 높임, 하이라이트 추가로 먹음직스럽게

| 국/찌개류 | 미역국: 진한 올리브-다크그린 국물 + 소고기 브라운. 김치찌개: 선명한 레드-오렌지 국물 + 김치. 콩나물국: 맑은 연노랑 국물 + 흰 콩나물 |

| 서브반찬 | 배추김치: 선명한 레드-코랄 + 배추 흰색. 쌈채소: 밝고 진한 그린 |

## 3. 캐릭터 스타일

### 3.1 전체 방향

- 동물의 숲 주민처럼 통통하고 단순한 형태
- 외곽선: 굵고 또렷 (다크 브라운 `#4A2E1A`, 2~3px)
- 음영: 셀 셰이딩 2단계 (기본색 + 그림자색 1단계)
- 눈: 크고 동그랗게, 하이라이트 1개, 검정 또는 다크브라운
- 코/입: 최소화 (코는 작은 점 또는 삼각형, 입은 짧은 곡선)
- 볼터치: 필수 (소프트 핑크 `#E8A0A0`, 반투명 원)
- 비율: 머리 : 몸 = 1 : 1 ~ 1 : 1.2 (머리가 큰 SD 비율)

### 3.2 일반 손님 4종 스타일 방향

| 캐릭터 | 주조색 | 특징 포인트 |
|--------|--------|-----------|
| 참새 | 브라운 + 아이보리 | 노란 부리, 작은 날개 |
| 쥐 | 미디엄 그레이 | 동그란 귀, 긴 꼬리 |
| 고양이 | 크림 / 오렌지 / 그레이 | 뾰족한 귀, 수염 |
| 강아지 | 베이지 / 브라운 | 처진 귀, 촉촉한 눈 |

### 3.3 에피소드 손님 스타일 방향

- 일반 손님보다 약간 크고 디테일 추가
- 개성 있는 소품 포함 (구상 중)
- 표정 변화 있음 (에피소드 진행에 따라 기본/기쁨/슬픔/놀람)

## 4. 배경/환경 스타일

### 4.1 식당 내부

- 원근감: 살짝 위에서 내려다보는 2.5D 뷰
- 외곽선: 캐릭터와 동일하게 굵은 선화 적용
- 벽/바닥: 따뜻한 나무 질감, 크림/베이지 계열, 셀 셰이딩
- 조명: 노란빛 따뜻한 포인트 조명 표현 (색상 오버레이)
- 소품: 작은 화분, 액자, 간판 등 아기자기한 요소 배치
- 채도: 캐릭터/음식보다 낮게 유지 (배경이 앞으로 튀지 않도록)

### 4.2 메인 게임 화면

- 조리기구 영역: 주방 느낌의 따뜻한 브라운 계열 배경
- 주문 리스트 영역: 화이트보드 또는 메모지 질감, 굵은 테두리
- 전체 배경: 게임 플레이 요소가 잘 보이도록 배경 채도 낮게 유지

## 5. UI 스타일

### 5.1 버튼

- 형태: 모서리가 둥근 라운드 렉탱글
- 외곽선: 다크 브라운 (`#4A2E1A`), 두께 2~3px
- 배경: 크림 또는 피치 계열 단색 + 하단 두꺼운 테두리로 입체감
- 눌림 효과: 하단 테두리 사라지며 아래로 2px 이동
- 텍스트: 다크 브라운, 볼드, 굵은 폰트

### 5.2 아이콘

- 선화 기반 단순 일러스트, 굵은 외곽선
- 내부 플랫 + 셀 셰이딩 1단계
- 크기 통일: 48×48px 기준 (UI 내 표시 크기)

### 5.3 폰트 방향

- 손글씨 느낌의 둥근 볼드 폰트
- 국문: 둥근 고딕 볼드 계열 (예: Nanum Round ExtraBold, 온글잎)
- 영문: 둥글고 두꺼운 손글씨 계열
- 크기 계층: 타이틀 32px / 본문 20px / 보조 16px (1920×1080 기준)

### 5.4 주문서 UI

- 메모지/영수증 질감의 아이보리 배경, 굵은 테두리
- 핀으로 꽂힌 듯한 연출
- 특별 손님 주문서: 연보라 (`#B0A0D8`) 배경으로 구분

## 6. 음식 아이콘 스타일

- 형태: 굵은 외곽선 (`#4A2E1A`, 2~3px) + 셀 셰이딩 2단계
- 실제 음식 색감에 가깝게, 하이라이트 1~2개로 먹음직스럽게 표현
- 그릇/접시: 연한 그레이-화이트 계열, 하이라이트 필수
- 음식별 특징 요소 1개씩 강조 (예: 미역국 — 진한 올리브 국물 + 소고기 + 파)
- 크기: 64×64px 기준 (조리기구 내 표시 크기)

### 6.1 음식별 스타일 메모

| 음식 | 그릇 | 주요 색상 | 강조 요소 |
|------|------|---------|---------|
| 미역국 | 흰 사발 | 다크 올리브그린 국물 | 소고기 조각, 파 |
| 김치찌개 | 뚝배기 | 선명한 레드오렌지 국물 | 김치, 두부, 파 |
| 콩나물국 | 흰 사발 | 맑은 연노랑 국물 | 콩나물, 고추 |
| 쌀밥 | 흰 공기 | 따뜻한 흰색 | 하이라이트, 살짝 보이는 밥알 |
| 잡곡밥 | 흰 공기 | 아이보리 + 브라운 스펙클 | 잡곡 알갱이 |
| 불고기 | 접시 | 글레이즈 다크브라운 | 파, 양파 |
| 생선구이 | 접시 | 골든 브라운 | 구운 자국, 레몬 |
| 닭갈비 | 접시 | 진한 오렌지레드 | 양념 색, 깻잎 |
| 배추김치 | 작은 접시 | 선명한 레드코랄 | 배추 흰 줄기 |
| 쌈채소 | 작은 접시 | 밝은 진한 그린 | 상추 결 |

## 7. 애니메이션 방향

| 요소 | 방향 |
|------|------|
| 캐릭터 대기 | 살랑살랑 좌우 흔들림, 0.8~1.0초 주기 |
| 조리 중 (냄비) | 뚜껑이 통통 튀는 애니메이션, 보글보글 이펙트 |
| 조리 중 (프라이팬) | 재료가 지글지글 흔들림, 연기 이펙트 |
| 조리 중 (밥솥) | 수증기 모락모락 |
| 조리 완성 | 반짝이는 별 이펙트 1~2개 |
| 탄 음식 | 검은 연기 이펙트 |
| 서빙 완료 | 하트 또는 별 1개 팝업 |
| 손님 퇴장 (화남) | 화남 이모티콘 팝업 + 빠른 퇴장 |
| 버튼 탭 | 살짝 눌리는 스케일 애니메이션 (0.95배) |
| 이징 | ease-out 기반, 통통 튀는 느낌은 bounce |

comic style illustration, thick outlines, cel shading, flat colors,
warm and cozy atmosphere, cute cartoon style, game asset,

cute [animal] character, chibi proportion, big round eyes,

warm colors, simple cartoon style, game character design

```
[food name] in a bowl, comic illustration style, thick dark outline,
cel shading, appetizing and vivid food colors, highlight on bowl,
cute cartoon food icon, game asset, white background

```
cozy restaurant interior, 2.5D isometric view, comic style,
thick outlines, cel shading, warm wood tones, beige walls,
