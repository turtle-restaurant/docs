## 1. 전체 비주얼 방향

### 1.1 키워드

```
귀여움 / 아기자기 / 깔끔한 선화 / 플랫 컬러 / 따뜻한 색감 / 스티커 느낌
```

### 1.2 레퍼런스

- **캐릭터**: 첨부 동물 스티커 이미지 — 굵은 다크브라운 외곽선, 단색 플랫 fill, 음영 없음, 스티커 컷아웃 스타일
- **음식**: 동일한 선화+플랫 컬러 방식으로 통일 — 수채화/번짐 없이, 캐릭터와 같은 그림체
- **배경/UI**: 캐릭터와 같은 선화 톤, 채도만 낮게 유지

### 1.3 한 줄 방향

> "귀엽고 아기자기한 스티커 그림체로 그린 따뜻한 한식 식당"

### 1.4 카테고리별 스타일 정의

| 카테고리 | 렌더링 방식        | 선화                | 음영                     |
| ---- | ------------- | ----------------- | ---------------------- |
| 캐릭터  | 플랫 컬러 + 굵은 선화 | 다크브라운, 굵게 (2~3px) | 없음 — 포인트 색 1~2개로만 구분   |
| 음식   | 플랫 컬러 + 굵은 선화 | 캐릭터와 동일           | 없음 — 음식 특징 디테일 선으로만 표현 |
| 배경   | 플랫 컬러 + 얇은 선화 | 캐릭터보다 얇게 (1~2px)  | 없음                     |
| UI   | 플랫 컬러 + 중간 선화 | 2px, 둥글게          | 버튼 하단 두꺼운 테두리로만 입체감    |

## 3. 캐릭터 스타일

> 레퍼런스: 첨부 동물 스티커 이미지 (여우/코알라/토끼/곰/판다/고양이/나무늘보/돼지/강아지)

### 3.1 렌더링 방식

- **선화**: 다크브라운 (`#3D2010`), 굵기 2~3px, 끊김 없이 매끄럽게
- **fill**: 단색 플랫 컬러 — 그라데이션/번짐/음영 전혀 없음
- **포인트 색**: 캐릭터당 최대 2개 (예: 귀 안쪽 핑크, 배 아이보리)
- **눈**: 검정 동그라미 fill + 흰 하이라이트 점 1개 (오른쪽 상단 고정)
- **코**: 작은 타원 또는 Y자형, 다크브라운
- **입**: 짧은 U자 또는 생략
- **볼터치**: Blush(`#F0B8B8`) 반투명 원, 눈 아래 좌우 대칭
- **스티커 컷아웃**: 캐릭터 외곽을 흰색 테두리(4~6px)로 감싸는 스티커 스타일

### 3.2 형태/비율 기준

- 비율: 머리 : 몸 = 1 : 0.8 ~ 1 : 1 (머리가 더 큰 SD)
- 체형: 통통하고 동글동글, 사지는 짧고 뭉툭하게
- 앉은 자세 기본 (레퍼런스 이미지 기준)
- 꼬리/날개/부리 등 동물 특징: 실루엣으로 읽힐 수 있는 최소 표현

### 3.3 일반 손님 4종

| 캐릭터 | 주조색 | 포인트 색 | 동물 특징 |
|--------|--------|----------|---------|
| 참새 | 브라운 `#A0724A` + 아이보리 `#F0E8D0` | 노란 부리 | 작고 단순한 날개, 둥근 몸 |
| 쥐 | 미디엄 그레이 `#B0A8A0` | 귀 안쪽 Blush | 동그란 귀 2개, 얇은 꼬리 선 |
| 고양이 | 오렌지 `#E8904A` (줄무늬 다크오렌지) | 귀 안쪽 Blush | 뾰족한 귀, 줄무늬 선 3개, 꼬리 |
| 강아지 | 베이지 `#D4A870` | 귀/코 다크브라운 | 처진 귀, 큰 눈, 짧은 꼬리 |

### 3.4 에피소드 손님 6종

### 3.5 주인공 거북이

- 동일한 스티커 스타일 유지
- 몸통: 등껍질 표현 (육각형 패턴 선으로 단순화), 그린 계열
- 얼굴: 손님 캐릭터와 동일한 눈/코/입/볼터치 규칙

## 4. 음식 이미지 스타일

> 캐릭터와 동일한 굵은 선화 + 플랫 컬러 방식으로 통일

### 4.1 렌더링 방식

- **선화**: 다크브라운 (`#3D2010`), 굵기 2~3px — 캐릭터와 동일
- **fill**: 단색 플랫 컬러 — 수채화/번짐/그라데이션 없음
- **디테일**: 음식 질감/특징은 얇은 선 몇 개로만 표현 (예: 밥알 점선, 고기 결 선)
- **하이라이트**: 흰 타원 또는 흰 점 1개 (그릇/국물 위에)
- **배경**: 흰색 또는 투명, 접시/그릇 포함 완성 일러스트

### 4.2 접시/그릇 형태 (GDD_game 메뉴 구성 기준)

| 카테고리 | 접시 형태 |
|----------|----------|
| 밥류 | 동그라미 공기, 뚜껑 없음, 돔형 밥 |
| 메인반찬 (불고기/생선구이/닭갈비) | 높이가 긴 직사각형 접시 |
| 국/찌개류 | 동그라미 그릇/뚝배기 |
| 서브반찬 | 1×2 분할 직사각형 접시 |

### 4.3 음식별 상세 메모

| 음식 | 접시/그릇 | 플랫 컬러 포인트 |
|------|----------|----------------|
| 쌀밥 | 흰 공기, 동그라미 | 아이보리 돔형, 흰 하이라이트 타원, 밥알 점선 |
| 잡곡밥 | 흰 공기, 동그라미 | 아이보리 베이스 + 브라운/다크 점 산포 |
| 미역국 | 흰 사발, 동그라미 | 올리브그린 국물 fill, 소고기 브라운 덩어리, 파 그린 점 |
| 김치찌개 | 뚝배기, 동그라미 | 레드오렌지 국물, 김치 레드 덩어리, 두부 흰 사각형 |
| 콩나물국 | 흰 사발, 동그라미 | 연노랑 국물, 흰 콩나물 선, 빨간 고추 점 |
| 불고기 | 직사각형 (높이긴) | 다크브라운 고기, 파 그린 선, 양파 흰 선 |

## 5. 배경/환경 스타일

### 5.1 전체 방향

캐릭터/음식과 동일한 선화 톤 유지. 배경이 앞으로 튀지 않도록 채도를 낮게 유지.

- **선화**: 다크브라운 (`#3D2010`), 굵기 1~2px (캐릭터보다 얇게)
- **fill**: 플랫 단색, 그라데이션 없음
- **채도**: 캐릭터/음식보다 30~40% 낮게

### 5.2 식당 내부

- 원근감: 살짝 위에서 내려다보는 2.5D 뷰
- 벽/바닥: 따뜻한 크림/베이지 계열 플랫 컬러, 나무 질감은 선으로만 단순화
- 조명: 노란빛 포인트 조명 — 원형 노란 영역 플랫으로 표현
- 소품: 화분, 액자, 간판 — 캐릭터와 같은 선화 굵기로 아기자기하게

### 5.3 메인 게임 화면

- 조리기구 영역: 따뜻한 브라운 계열 플랫 배경
- 주문 리스트 영역: 아이보리 플랫 배경, 다크브라운 굵은 테두리
- 전체: 채도 낮게 유지, 캐릭터/음식/UI가 도드라지도록

## 6. UI 스타일

### 6.1 전체 방향

캐릭터와 동일한 굵은 선화 + 플랫 컬러. 종이/메모지 느낌보다는 깔끔한 플랫 카드 스타일.

### 6.2 버튼

- 형태: 모서리 둥근 라운드 렉탱글
- 외곽선: Cocoa (`#3D2010`), 2~3px
- 배경: 크림 또는 피치 계열 단색
- 입체감: 하단에 다크 컬러 두꺼운 테두리 (shadow 느낌, 그라데이션 없음)
- 눌림 효과: 하단 테두리 사라지며 2px 아래로 이동

### 6.3 아이콘

- 선화 기반 단순 일러스트, 캐릭터와 동일한 굵기
- 내부 플랫 단색
- 크기 통일: 48×48px 기준

### 6.4 폰트

- 국문: 둥근 고딕 볼드 계열 (예: Nanum Round ExtraBold, 온글잎)
- 영문: 둥글고 두꺼운 손글씨 계열
- 크기 계층: 타이틀 32px / 본문 20px / 보조 16px (1920×1080 기준)

### 6.5 주문서 UI

- 아이보리 플랫 배경, 다크브라운 굵은 테두리
- 핀으로 꽂힌 듯한 연출 (핀 아이콘, 플랫 컬러)
- 특별 손님 주문서: Lavender (`#B0A0D8`) 배경으로 구분
