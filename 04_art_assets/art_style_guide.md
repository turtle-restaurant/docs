## 3. 캐릭터 스타일

> 레퍼런스: `ref_charactor_common_guests.png`, `ref_charactor_turtle.png`

### 3.1 공통 렌더링 방식

- **선화**: 다크브라운 (`#3D2010`), 굵고 매끄럽게, 끊김 없이
- **채색**: 영역별 단색 베이스 — 그라데이션 없음. 밝은 하이라이트 1~2곳으로만 입체감 표현
- **눈**: 크고 동그란 검정 눈동자, 흰 하이라이트 점 1개 (우상단 고정)
- **코**: 작은 타원 또는 동그란 코, 다크브라운
- **입**: 짧은 U자형 미소
- **볼터치**: Blush(`#F0B8B8`) 반투명 원, 눈 아래 좌우 대칭
- **배 부분**: 아이보리 계열 밝은 타원으로 구분 (레퍼런스 기준)

### 3.2 형태/비율 기준

- 비율: 머리 : 몸 = 1 : 1 ~ 1 : 1.2 (약간 머리 큰 SD 비율)
- 체형: 통통하고 동글동글, 팔다리 짧고 뭉툭하게
- **자세: 스탠딩 정면, 두 발로 서서 정면을 응시하는 기본 자세**
- 꼬리/날개/수염 등 동물 특징: 실루엣으로 읽히는 최소 표현

### 3.3 일반 손님 4종

| 캐릭터 | 주조색 | 포인트 색 | 동물 특징 |
|--------|--------|----------|---------|
| 참새 | 브라운 `#A0724A` + 아이보리 배 `#F0E8D0` | 노란 부리 | 단순한 날개, 둥근 몸 |
| 쥐 | 미디엄 그레이 `#B0A8A0` + 아이보리 배 | 귀 안쪽 Blush | 동그란 귀 2개, 가는 꼬리 |
| 고양이 | 골든 오렌지 `#E8B84A` + 줄무늬 | 귀 안쪽 Blush, 아이보리 배 | 뾰족한 귀, 줄무늬 3개, 꼬리 |
| 강아지 | 베이지 `#D4A870` + 아이보리 배 | 처진 귀 다크브라운 | 처진 귀, 큰 눈, 짧은 꼬리 |

### 3.4 에피소드 손님 6종

| 캐릭터 | 주조색 | 포인트 색 | 동물 특징 |

| 곰 | 브라운 `#8B5030` + 아이보리 배 | 코 다크브라운 | 동그란 귀, 통통한 발 |
| 너구리 | 그레이 `#9090A0` + 아이보리 배 | 검정 눈 주변 마스크 | 굵은 꼬리, 눈 주변 다크 패치 |

### 3.5 주인공 거북이

> 레퍼런스: `ref_charactor_turtle.png`

- 화풍/색감/의상은 레퍼런스 이미지 기준으로 유지
- 몸통: 등껍질 (브라운 계열 `#8B6040`, 육각형 패턴), 피부 연두 `#88C060`
- 얼굴: 손님 캐릭터와 동일한 눈/코/입/볼터치 규칙
- 셰프 복장: 흰 셰프 모자 + 흰 앞치마 (레퍼런스 기준)
- **자세: 스탠딩 정면 기본. 포즈/표정은 필요에 따라 별도 생성**
- 표정 5종 필요: 기본 / 요리중 (집중) / 서빙중 (기쁨) / 당황 / 피곤

## 5. 배경/환경 스타일

### 5.1 전체 방향

캐릭터/음식이 전면에 도드라지도록 배경은 채도와 디테일을 낮게 유지.

- **선화**: 다크브라운 (`#3D2010`), 굵기 1~1.5px (캐릭터보다 얇게)
- **채색**: 단순 플랫 컬러, 그라데이션 없음
- **채도**: 캐릭터/음식 대비 30~40% 낮게

### 5.2 식당 내부

- 원근감: 살짝 위에서 내려다보는 2.5D 뷰
- 벽/바닥: 따뜻한 크림/베이지 계열, 나무 질감은 선으로만 단순화
- 조명: 노란빛 원형 포인트, 플랫으로 표현
- 소품: 화분, 액자, 간판 — 아기자기한 선화로 표현

### 5.3 메인 게임 화면

- 조리기구 영역: 따뜻한 브라운 계열 배경
- 주문 리스트 영역: 아이보리 배경, 다크브라운 테두리
- 전체: 캐릭터/음식/UI가 도드라지도록 배경은 억제

## 6. UI 스타일

### 6.1 전체 방향

굵은 선화 + 플랫 컬러. 깔끔한 플랫 카드 스타일.

### 6.2 버튼

- 형태: 모서리 둥근 라운드 렉탱글
- 외곽선: Cocoa (`#3D2010`), 2~3px
- 배경: 크림 또는 피치 계열 단색
- 입체감: 하단 다크 컬러 두꺼운 테두리 (그라데이션 없음)
- 눌림 효과: 하단 테두리 사라지며 2px 아래 이동

### 6.3 아이콘

- 선화 기반 단순 일러스트, 캐릭터와 동일 선 굵기
- 내부 플랫 단색
- 크기 통일: 48×48px 기준

### 6.4 폰트

- 국문: 둥근 고딕 볼드 계열 (예: Nanum Round ExtraBold, 온글잎)
- 영문: 둥글고 두꺼운 손글씨 계열
- 크기 계층: 타이틀 32px / 본문 20px / 보조 16px (1920×1080 기준)

### 6.5 주문서 UI

- 아이보리 배경, 다크브라운 굵은 테두리
- 핀으로 꽂힌 듯한 연출 (핀 아이콘, 플랫 컬러)
- 특별 손님 주문서: Lavender (`#B0A0D8`) 배경으로 구분

## 7. 애니메이션 방향

| 요소 | 방향 |
|------|------|
| 캐릭터 대기 | 살랑살랑 좌우 흔들림, 0.8~1.0초 주기 |
| 조리 중 (냄비) | 뚜껑 통통 튀는 애니메이션, 보글보글 이펙트 |
| 조리 중 (프라이팬) | 재료 지글지글 흔들림, 연기 이펙트 |
| 조리 중 (밥솥) | 수증기 모락모락 |
| 조리 완성 | 반짝이는 별 이펙트 1~2개 |
| 탄 음식 | 검은 연기 이펙트 |
| 서빙 완료 | 하트 또는 별 1개 팝업 |
| 손님 퇴장 (화남) | 화남 이모티콘 팝업 + 빠른 퇴장 |
| 버튼 탭 | 스케일 0.95배 눌림 효과 |
| 이징 | ease-out 기반, 통통 튀는 느낌은 bounce |
