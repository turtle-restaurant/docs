## 1. 전체 비주얼 방향

### 1.1 키워드

```
따뜻함 / 귀여움 / 아기자기 / 파스텔 / 손그림 / 단순하고 부드러운
```

### 1.2 레퍼런스

- 첨부 이미지 (토끼 요리 게임) — 전체적인 아기자기함과 캐릭터 밀도 참고
- 단, 본 게임은 위 레퍼런스보다 **더 연한 파스텔톤** 적용
- 음식은 파스텔톤이되 **먹음직스러운 채도와 디테일** 유지

### 1.3 한 줄 방향

> "연한 파스텔 크레파스로 그린 듯한 따뜻한 손그림 식당"

### 2.4 음식 색상 방향

파스텔톤을 유지하되, 음식만큼은 채도를 10~15% 높여 먹음직스럽게 표현.

| 음식 카테고리 | 색상 방향 |
|-------------|---------|
| 밥류 | 따뜻한 아이보리 (`#F5EDD0`), 잡곡은 연보라 스펙클 추가 |
| 국/찌개류 | 김치찌개 연한 주황-빨강, 미역국 진한 올리브 브라운, 콩나물국 맑은 연노랑 |
| 메인반찬 | 불고기 글레이즈 브라운, 생선구이 골든 베이지, 닭갈비 주황빛 소스 |
| 서브반찬 | 배추김치 선명한 코랄-핑크, 쌈채소 밝은 그린 |

## 3. 캐릭터 스타일

### 3.1 전체 방향

- 동물의 숲 주민처럼 통통하고 단순한 형태
- 눈: 크고 동그랗게, 하이라이트 1개
- 코/입: 최소화 (코는 작은 점 또는 삼각형, 입은 짧은 곡선)
- 볼터치: 필수 (소프트 핑크 `#F2C4C4`)
- 선: 외곽선 있음, 두께 일정 (캐릭터 크기 대비 2~3px 수준)
- 음영: 단순 플랫 + 한 단계 어두운 그림자만 사용 (복잡한 그라데이션 없음)
- 비율: 머리 : 몸 = 1 : 1 ~ 1 : 1.2 (머리가 큰 SD 비율)

### 3.2 일반 손님 4종 스타일 방향

| 캐릭터 | 주조색 | 특징 포인트 |
|--------|--------|-----------|
| 참새 | 소프트 브라운 + 아이보리 | 노란 부리, 작은 날개 |
| 쥐 | 라이트 그레이 | 동그란 귀, 긴 꼬리 |
| 고양이 | 크림 화이트 / 오렌지 / 그레이 | 뾰족한 귀, 수염 |
| 강아지 | 소프트 베이지 / 브라운 | 처진 귀, 촉촉한 눈 |

### 3.3 에피소드 손님 스타일 방향

일반 손님보다 약간 크고 디테일 추가. 개성 있는 소품 포함 (구상 중).

## 4. 배경/환경 스타일

### 4.1 식당 내부

- 원근감: 등각 투영(isometric) 또는 살짝 위에서 내려다보는 2.5D 뷰
- 벽/바닥: 따뜻한 나무 질감, 크림/베이지 계열
- 조명: 노란빛 따뜻한 포인트 조명, 창문으로 들어오는 햇빛 연출
- 소품: 작은 화분, 액자, 간판 등 아기자기한 요소 배치
- 채도: 전체적으로 낮게 유지, 조명 부근만 살짝 높임

### 4.2 메인 게임 화면

- 조리기구 영역: 주방 느낌의 따뜻한 브라운 계열 배경
- 주문 리스트 영역: 화이트보드 또는 메모지 질감
- 전체 배경: 복잡하지 않게, 게임 플레이 요소가 잘 보이도록 단순하게 유지

## 5. UI 스타일

### 5.1 버튼

- 형태: 모서리가 많이 둥근 라운드 렉탱글
- 외곽선: 카라멜 브라운 (`#C8845A`), 두께 2px
- 배경: 크림 또는 피치 계열 단색
- 눌림 효과: 살짝 아래로 이동 + 약간 어두워짐
- 텍스트: 브라운 계열, 볼드

### 5.2 아이콘

- 선화 기반 단순 일러스트
- 외곽선 있음, 내부 플랫 채색
- 크기 통일: 48×48px 기준 (UI 내 표시 크기)

### 5.3 폰트 방향

- 손글씨 느낌의 둥근 폰트
- 국문: 둥근 고딕 계열 (예: Nanum Round, 온글잎 등)
- 영문: 둥글고 두꺼운 손글씨 계열
- 크기 계층: 타이틀 32px / 본문 20px / 보조 16px (1920×1080 기준)

### 5.4 주문서 UI

- 메모지/영수증 질감의 아이보리 배경
- 핀으로 꽂힌 듯한 연출
- 특별 손님 주문서: 연보라(`#D4C4E8`) 또는 연핑크(`#F2C4C4`) 배경으로 구분

---

## 6. 음식 아이콘 스타일

- 형태: 단순 일러스트, 외곽선 있음
- 그림자: 아이콘 하단에 작은 드롭 섀도우 1개
- 디테일: 파스텔 베이스 + 하이라이트 1~2개로 입체감 표현
- 음식별 특징 요소 1개씩 강조 (예: 김치찌개 - 빨간 국물 + 김치 한 조각)
- 크기: 64×64px 기준 (조리기구 내 표시 크기)

## 7. 애니메이션 방향

| 요소 | 방향 |
|------|------|
| 캐릭터 대기 | 살랑살랑 좌우 흔들림, 0.8~1.0초 주기 |
| 조리 중 (냄비) | 뚜껑이 통통 튀는 애니메이션, 보글보글 이펙트 |
| 조리 중 (프라이팬) | 재료가 지글지글 흔들림, 연기 이펙트 |
| 조리 중 (밥솥) | 수증기 모락모락 |
| 조리 완성 | 반짝이는 별 이펙트 1~2개 |
| 탄 음식 | 검은 연기 이펙트 |
| 서빙 완료 | 하트 또는 별 1개 팝업 |
| 손님 퇴장 (화남) | 화남 이모티콘 팝업 + 빠른 퇴장 |
| 버튼 탭 | 살짝 눌리는 스케일 애니메이션 (0.95배) |
| 이징 | 전반적으로 ease-out 기반, 통통 튀는 느낌은 bounce |
