# 아트 방향성 가이드

> 작성일: 2025-05-19  
> 담당: 아트 에이전트 (Claude.ai + Midjourney)

## 아트 스타일 정의

| 항목 | 방향 |
|------|------|
| 전체 톤 | 파스텔 + 스케치톤 |
| 질감 | 손으로 그린 듯한 연필/수채화 느낌 |
| 색감 | 따뜻하고 부드러운 파스텔 팔레트 |
| 캐릭터 | 둥글고 귀여운 동물, 2등신~2.5등신 |
| UI | 종이 질감 배경, 손글씨 느낌 폰트 |

---

## Midjourney 기본 프롬프트 공식

```
[캐릭터/오브젝트 설명], cute pastel illustration, sketch style, 
soft watercolor texture, warm color palette, 2D game asset, 
white background, Studio Ghibli inspired, cozy healing atmosphere
--ar [비율] --v 6
```

### 예시: 거북이 주인공
```
cute turtle chef character, wearing apron, happy expression, 
pastel illustration, sketch style, soft watercolor texture, 
warm color palette, 2D game asset, white background, 
Studio Ghibli inspired --ar 1:1 --v 6
```
