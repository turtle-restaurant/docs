## 1. 게임 개요

### 1.1 핵심 경험

날짜별로 점심·저녁 식당을 운영하며 밀려드는 주문을 처리하고,
개성 있는 단골 손님들을 하나씩 해금하며 힐링 식당 이야기를 즐기는 게임.

### 1.2 장르 및 타겟

- **장르**: 힐링 아케이드 + 운영 시뮬레이션
- **플레이 방향**: 가로형
- **해상도**: 1920×1080 기준 (환경별 유연 대응, 모바일 가로 포함)
- **타겟**: 캐주얼 게이머, 힐링 게임 선호 20~30대
- **레퍼런스**: 쿠킹 매드니스, 네모네모 스폰지밥: 집게리아 요리 콘테스트, 동물의 숲(캐릭터 감성)

### 1.3 아트 방향

- 파스텔 + 스케치톤 (손으로 그린 듯한 느낌)
- 동물의 숲 주민처럼 국가/문화 구분 없는 공통 동물 캐릭터
- 따뜻하고 부드러운 색감
- 조리 이미지: 보글보글 끓는 냄비, 굽는 프라이팬 등 디테일한 애니메이션 에셋으로 제작

### 1.4 게임 구조 전체도

```
거북이식당
├── 메인 스토리 (기본 모드)
│   ├── 날짜별 진행
│   │   ├── 1회차: 점심 오픈 (보통 난이도)
│   │   └── 2회차: 저녁 오픈 (어려움 난이도, 스킵 가능)
│   ├── 기본 게임: 주문 처리 + 조리 + 서빙
│   ├── 날짜 달성 + 조리기구 구매 → 특별 손님 해금 / 에피소드 오픈
│   ├── 에피소드 완료 손님 → 이후 날짜부터 일반 손님 풀에 합류
│   │   └── 방문 시 보너스 골드 + 확률 아이템/스킨 + 방문률 상승 인테리어 아이템
│   └── 지난 날짜 리플레이 가능 (달력 메뉴)
│       ├── 첫 클리어: 골드 + 젬
│       └── 리플레이: 골드만 지급
│
└── 시즌 축제 (별도 메뉴)
    ├── 샌드위치 스택 미니게임
    ├── 재료 슬라이서 미니게임
    ├── 레시피 메모리 미니게임
    └── 시즌 랭킹 (골드 + 젬 보상)
```

### 1.5 게임 프로세스

```
[타이틀 화면]
└── 게임 시작 버튼
    └── [메인 화면]
        ├── 상단 HUD
        │   ├── 현재 날짜
        │   ├── 달력 아이콘 (지난 날짜 리플레이 메뉴)
        │   ├── 재화 표시 (골드, 젬)
        │   └── 상점 / 설정 아이콘
        ├── 배경: 식당 메인 홀
        └── 우측 버튼
            ├── [오픈하기] → 메인 게임 진입
            │   ├── 1회차: 점심 오픈 / 2회차: 저녁 오픈
            │   ├── 날짜+시간대 기준 해금 특별 손님 에피소드 영상 (스킵 가능)
            │   ├── 게임 시작 — 현재 날짜 + 점심/저녁 페이드인-페이드아웃 노출
            │   ├── 메인 게임 진행
            │   └── 게임 완료 팝업
            │       ├── 획득 재화 / 손님 수 표시
            │       └── 이전 / 재시도 / 다음 단계 버튼
            ├── [창고] → 업그레이드 및 인테리어
            │   ├── 주방용품 업그레이드 (조리기구 항목별, 골드)
            │   └── 홀 인테리어 (벽지, 바닥 타일, 가구, 장식)
            ├── [일기] → 기록 메뉴
            │   ├── 누적 손님 수 기록
            │   └── 특별 손님 에피소드 영상 회차별 재감상
            └── [시즌 축제] → 축제 메뉴
                ├── 시즌 정보 (테마명, 기간, 랭킹별 보상)
                ├── 미니게임 3종 버튼
                └── 랭킹 / 기록 확인 버튼
```

### 1.7 특별 손님 해금 조건

- 해금 조건: 지정 날짜 달성 AND 지정 조리기구 구매 완료
- 조리기구는 골드로만 구매 가능 (BM 비연결)
- 두 조건 중 하나라도 미충족 시 해금 불가
- 조리기구 구매 완료 후 날짜 미달 시 → 날짜 달성 시점에 자동 해금
- 날짜 달성 후 조리기구 미구매 시 → 구매 완료 시점에 자동 해금
