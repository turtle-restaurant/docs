## 1. 게임 개요

### 1.1 핵심 경험

날짜별로 점심·저녁 식당을 운영하며 개성 있는 단골 손님들을 하나씩 해금하고,
시즌 축제 이벤트에서는 특별 미니게임 챌린지를 즐길 수 있는 힐링 식당 게임.

### 1.2 장르 및 타겟

- **장르**: 힐링 아케이드 + 운영 시뮬레이션
- **플레이 방향**: 가로형
- **해상도**: 1334×750 (iPhone 8 기준, 16:9 비율 모바일 가로형 표준)
- **타겟**: 캐주얼 게이머, 힐링 게임 선호 20~30대
- **레퍼런스**: 쿠킹 매드니스, 네모네모 스폰지밥: 집게리아 요리 콘테스트, 동물의 숲(캐릭터 감성)

### 1.3 아트 방향

- 파스텔 + 스케치톤 (손으로 그린 듯한 느낌)
- 동물의 숲 주민처럼 국가/문화 구분 없는 공통 동물 캐릭터
- 따뜻하고 부드러운 색감

### 1.4 게임 구조 전체도

```
거북이식당
├── 메인 스토리 (기본 모드)
│   ├── 날짜별 진행
│   │   ├── 1회차: 점심 오픈 (보통 난이도)
│   │   └── 2회차: 저녁 오픈 (어려움 난이도, 스킵 가능)
│   ├── 기본 게임: 타이밍 쿡
│   ├── 날짜 달성 + 조리기구 구매 → 특별 손님 해금 / 에피소드 오픈
│   ├── 에피소드 완료 손님 → 이후 날짜부터 일반 손님 풀에 합류
│   │   └── 방문 시 보너스 골드 + 확률 아이템/스킨 + 방문률 상승 인테리어 아이템
│   └── 지난 날짜 리플레이 가능 (달력 메뉴)
│       ├── 첫 클리어: 골드 + 젬
│       └── 리플레이: 골드만 지급
│
└── 시즌 축제 (별도 메뉴)
    ├── 샌드위치 스택 미니게임
    ├── 재료 슬라이서 미니게임
    ├── 레시피 메모리 미니게임
    └── 시즌 랭킹 (골드 + 젬 보상)
```

### 1.5 게임 프로세스

```
[타이틀 화면]
└── 게임 시작 버튼
    └── [메인 화면]
        ├── 상단 HUD
        │   ├── 현재 날짜
        │   ├── 달력 아이콘 (지난 날짜 리플레이 메뉴)
        │   ├── 재화 표시 (골드, 젬)
        │   └── 상점 / 설정 아이콘
        ├── 배경: 식당 메인 홀
        └── 우측 버튼
            ├── [오픈하기] → 메인 게임 진입
            │   ├── 1회차: 점심 오픈 / 2회차: 저녁 오픈
            │   ├── 날짜+시간대 기준 미해금 특별 손님 에피소드 영상 (스킵 가능)
            │   ├── 게임 시작 — 현재 날짜 + 점심/저녁 페이드인-페이드아웃 노출
            │   ├── 메인 게임 진행 (타이밍 쿡)
            │   └── 게임 완료 팝업
            │       ├── 획득 재화 / 손님 수 표시
            │       └── 이전 / 재시도 / 다음 단계 버튼
            ├── [창고] → 업그레이드 및 인테리어
            │   ├── 주방용품 업그레이드
            │   │   └── 조리기구 항목별 업그레이드 (골드)
            │   └── 홀 인테리어
            │       ├── 벽지 / 바닥 타일
            │       ├── 가구 (테이블, 의자)
            │       └── 장식 (조명, 식물, 간판 등)
            ├── [일기] → 기록 메뉴
            │   ├── 누적 손님 수 기록
            │   └── 특별 손님 에피소드 영상 회차별 재감상
            └── [시즌 축제] → 축제 메뉴
                ├── 시즌 정보 (테마명, 기간, 랭킹별 보상)
                ├── 미니게임 3종 버튼 (샌드위치 스택 / 재료 슬라이서 / 레시피 메모리)
                └── 랭킹 / 기록 확인 버튼
```

### 1.6 보상 구조 요약

| 플레이 유형 | 골드 | 젬 | 드롭 아이템 |
|-------------|------|----|------------|
| 점심 첫 클리어 | O | O | - |
| 저녁 첫 클리어 | O | O | - |
| 리플레이 (점심/저녁 무관) | O | X | X |
| 특별 손님 방문 | O (1.5~2배) | X | O (10~20% 확률) |
| 시즌 랭킹 보상 | O | O | - |

### 1.7 특별 손님 해금 조건

- 해금 조건: 지정 날짜 달성 **AND** 지정 조리기구 구매 완료
- 조리기구는 골드로만 구매 가능 (BM 비연결)
- 두 조건 중 하나라도 미충족 시 해금 불가
- 조리기구 구매 완료 후 날짜 미달 시 → 날짜 달성 시점에 자동 해금
- 날짜 달성 후 조리기구 미구매 시 → 구매 완료 시점에 자동 해금

## 2. 메인 스토리 — 타이밍 쿡

### 2.1 기본 흐름

```
손님 입장 → 주문 접수 → 타이밍 쿡 → 서빙 → 평가 → 골드 획득
```

### 2.2 타이밍 쿡 규칙

- 조리 단계가 순서대로 화면에 표시됨
- 각 단계마다 타이밍 바가 움직이고, 적절한 타이밍에 탭/클릭
- Perfect / Good / Miss 3단계 판정
- 판정 결과 → 음식 품질(★) 결정 → 골드 보상 배율에 영향

### 2.3 스테이지 진행 구조

| 단계 | 내용 | 클리어 조건 | 해금 보상 |
|------|------|------------|-----------|
| Stage 1 | 기본 레시피 3종, 튜토리얼 | 누적 매출 목표 달성 | 호랑님 해금 |
| Stage 2 | 레시피 추가, 타이밍 난이도 상승 | 누적 매출 목표 달성 | 솜이·코코 해금 |
| Stage 3 | 복합 조리 단계 등장 | 누적 매출 목표 달성 | 베어·타누키 해금 |
| Stage 4+ | 동시 주문 처리 추가 | 누적 매출 목표 달성 | 판판 해금 |

## 3. 손님 시스템

### 3.1 일반 손님

- 날짜에 맞는 랜덤 동물 손님 방문
- 국가 구분 없이 동물의 숲 스타일 공통 캐릭터
- 인내심, 선호 메뉴 등 개별 속성 보유

### 3.2 에피소드 손님 (해금형 단골)

- 지정 날짜 달성 + 지정 조리기구 구매 시 순차 해금
- 에피소드 완료 후 일반 손님 풀에 합류하여 낮은 빈도로 랜덤 방문
- **방문 시 혜택**
  - 일반 손님 대비 골드 1.5~2배 보상
  - 10~20% 확률로 특별 아이템 또는 스킨 드롭
  - 에피소드 진행 중: 방문마다 스토리 1회차 진행 (총 3~5회차)
- **에피소드 완료 보상**
  - 해당 손님 방문률 상승 인테리어 아이템 지급

### 3.3 손님 목록

| 이름 | 동물 | 해금 날짜 | 에피소드 키워드 | 드롭 보상 |
|------|------|-----------|----------------|-----------|
| 호랑님 | 호랑이 | Day 1 | 퇴직 후 새 출발 | 주방장 앞치마 스킨 |
| 솜이 | 토끼 | Day 2 | 수험생의 하루 | 행운의 냄비 |
| 코코 | 여우 | Day 2 | 첫 독립 자취 | 파스텔 식기 스킨 |
| 베어 | 곰 | Day 3 | 다이어트 중인 대식가 | 대형 프라이팬 |
| 타누키 | 너구리 | Day 3 | 고향이 그리운 여행자 | 여행자 모자 스킨 |
| 판판 | 판다 | Day 4 | 워커홀릭의 휴식 | 골드 국자 스킨 |

## 4. 시즌 축제 (별도 메뉴)

### 4.1 구조

- 메인 화면에서 별도 메뉴로 진입
- 시즌별 테마 변경 (봄 축제, 여름 축제, 할로윈, 크리스마스 등)
- 3종 미니게임 모두 플레이 가능

### 4.2 축제 미니게임 3종

| # | 이름 | 설명 |
|---|------|------|
| 1 | 샌드위치 스택 | 재료를 순서대로 드래그해서 쌓기, 흔들림 요소로 난이도 조절 |
| 2 | 재료 슬라이서 | 날아오는 재료를 타이밍에 맞춰 슬라이드, 연속 콤보 보너스 |
| 3 | 레시피 메모리 | 순서 보여준 뒤 기억해서 재현, 단계 길어질수록 난이도 상승 |

### 4.3 랭킹 시스템

- 3종 합산 점수로 시즌 랭킹 산정
- 시즌 종료 시 순위별 스킨·젬 보상
- 랭킹 구간: 일반 / 어려움 / 챌린저

## 5. 재화 & 경제 시스템

### 5.1 재화 종류

| 재화 | 획득 방법 | 사용처 |
|------|-----------|--------|
| 골드 | 서빙 보상, 에피소드 손님 보너스, 리플레이 | 인테리어, 레시피, 조리도구 |
| 젬 | 첫 클리어 보상, 시즌 랭킹 보상, IAP | 프리미엄 스킨, 특별 인테리어 |

### 5.2 상점 카테고리

- 인테리어 (테이블, 벽지, 소품) — 골드 / 일부 젬
- 조리도구 업그레이드 (판정 범위 확대, 속도 향상) — 골드 전용
- 레시피 오픈 (골드로 선구매 가능)
- 캐릭터 스킨 — 젬 구매 또는 에피소드 드롭

## 6. BM (비즈니스 모델)

- 기본 게임 무료
- 젬 패키지 IAP
- 캐릭터 스킨 번들 직접 구매
- 조리도구 스킨 (기능 동일, 외형 변경)
- 조리도구 기능 업그레이드는 골드 전용 (BM 비연결)
- 에피소드 드롭 아이템은 플레이 전용 보상 (BM과 별도)
