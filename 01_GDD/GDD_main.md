## 1. 게임 개요

### 1.1 핵심 경험
플레이어는 귀여운 거북이 주인이 되어 다양한 동물 손님들에게 음식을 만들어 대접한다.
바쁘지만 따뜻한 식당 운영의 즐거움과, 손님들과의 짧은 에피소드가 힐링감을 제공한다.

### 1.2 장르 및 타겟
- **장르**: 힐링 아케이드 + 운영 시뮬레이션
- **타겟**: 캐주얼 게이머, 힐링 게임 선호 20~30대
- **레퍼런스**: 쿠킹 마더, 캐피 테일즈, 파파스 프리즈리아

### 1.3 아트 방향
- 파스텔 + 스케치톤 (손으로 그린 듯한 느낌)
- 따뜻하고 부드러운 색감
- 동물 캐릭터: 각 문화권 특색 반영

---

## 2. 핵심 게임 루프

```
하루 시작
  → 컨셉데이 공지 (한식 / 양식 / 일식 / 중식)
  → 손님 순차 방문
     → 주문 접수
     → 미니게임으로 조리
     → 서빙 + 평가
     → 재화 획득
  → 하루 마감 (매출 / 손님 수 정산)
  → 상점에서 업그레이드
  → 다음 날로
```

## 3. 컨셉데이 & 미니게임

### 3.1 컨셉데이 종류
| 컨셉데이 | 테마    | 대표 손님       |
| ---- | ----- | ----------- |
| 한식데이 | 한국 음식 | 호랑이, 토끼, 까치 |
| 양식데이 | 서양 음식 | 여우, 곰, 다람쥐  |
| 일식데이 | 일본 음식 | 너구리, 사슴, 잉어 |
| 중식데이 | 중국 음식 | 판다, 용, 원숭이  |

### 3.2 미니게임 4종
| #   | 미니게임명       | 설명                     | 컨셉 연결   |
| --- | ----------- | ---------------------- | ------- |
| 1   | **샌드위치 스택** | 재료를 순서대로 쌓기 (드래그 앤 드롭) | 양식데이    |
| 2   | **타이밍 쿡**   | 타이밍 바에 맞춰 순서대로 조리      | 전체 공통   |
| 3   | **재료 슬라이서** | 재료를 리듬에 맞춰 슬라이드/탭      | 일식데이    |
| 4   | **레시피 메모리** | 레시피 순서 기억 후 재현         | 한식/중식데이 |

> 컨셉데이에 따라 4종 중 랜덤으로 1~2개 등장

## 4. 손님 & 에피소드 시스템

### 4.1 구조
- 손님별 고유 에피소드 (3~5회 방문 시 완결)
- 에피소드 클리어 → 전용 레시피 또는 아이템 보상
- 손님 호감도 시스템 (단골 손님 개념)

### 4.2 손님 목록 (초안)
| 이름 | 동물 | 문화권 | 에피소드 키워드 |
|------|------|--------|----------------|
| 호랑님 | 호랑이 | 한식 | 퇴직 후 새 출발 |
| 솜이 | 토끼 | 한식 | 수험생의 하루 |
| 코코 | 여우 | 양식 | 첫 독립 자취 |
| 베어 | 곰 | 양식 | 다이어트 중인 대식가 |
| 타누키 | 너구리 | 일식 | 고향이 그리운 여행자 |
| 판판 | 판다 | 중식 | 워커홀릭의 휴식 |

## 5. 재화 & 경제 시스템

## 6. 진행 방식

- **일별 진행**: Day 1, Day 2... 형식
- **정보 표시**: 날짜, 누적 매출, 오늘 손님 수, 만족도
- **난이도**:
  - 일반: 시간 여유, 실수 허용
  - 어려움: 타이트한 시간, 손님 인내심 감소
  - 미니게임 챌린저: 미니게임 난이도 극대화
