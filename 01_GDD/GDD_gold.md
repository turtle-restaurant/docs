## 1. 재화 종류

| 재화 | 획득 방법 | 사용처 |
|------|-----------|--------|
| 골드 | 서빙 보상, 에피소드 손님 보너스, 리플레이 | 인테리어, 레시피, 조리도구, 1회성 아이템 |
| 젬 | 첫 클리어 보상, 시즌 랭킹 보상, IAP | 프리미엄 스킨, 특별 인테리어 |

## 2. 보상 구조

| 플레이 유형 | 골드 | 젬 | 드롭 아이템 |
|-------------|------|----|------------|
| 점심 첫 클리어 | O | O | - |
| 저녁 첫 클리어 | O | O | - |
| 리플레이 | O | X | X |
| 특별 손님 방문 | O (1.5~2배) | X | O (10~20% 확률) |
| 시즌 랭킹 보상 | O | O | - |

## 3. 창고 (업그레이드 & 인테리어)

### 3.1 주방용품 업그레이드 (골드 전용)

| 항목 | 업그레이드 효과 |
|------|---------------|
| 냄비 | 조리 속도 증가 |
| 냄비 재료박스 | 골드 수급량 증가 |
| 프라이팬 | 조리 속도 증가 |
| 프라이팬 재료박스 | 골드 수급량 증가 |
| 밥솥 | 조리 속도 증가 |
| 쌀통 | 골드 수급량 증가 |
| 반찬박스 | 골드 수급량 증가 |
| 쟁반 추가 | 동시 서빙 가능 쟁반 수 증가 |

### 3.2 홀 인테리어

| 카테고리 | 항목 | 재화 |
|----------|------|------|
| 바닥/벽 | 벽지, 바닥 타일 | 골드 / 일부 젬 |
| 가구 | 테이블, 의자 | 골드 / 일부 젬 |
| 장식 | 조명, 식물, 간판 | 골드 / 일부 젬 |
| 특별 손님 방문률 상승 아이템 | 에피소드 완료 보상으로 지급 | - |

---

## 4. 골드 밸런스 방향

- 날짜가 높아질수록 기본 골드 수급량 증가 → 최신 날짜 플레이가 가장 효율적
- 리플레이는 골드만 지급 → 파밍 가능하나 젬/드롭 없음
- 특별 손님 해금 조건 조리기구는 골드로만 구매 → 페이월 없음
- 세부 수치는 레벨 디자인 시 확정 예정
