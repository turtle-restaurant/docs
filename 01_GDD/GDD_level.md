## 설계 전제

- 한 세션(점심 또는 저녁) 목표 플레이 시간: 1~2분
- 조리 기본 3~4초, 업그레이드 최대 1초 (쿠킹 매드니스 기준)
- 플레이어는 항상 동시 조리한다고 가정
- 주문 최대 건수 = 동시 화면 표시 상한 (처리되면 즉시 신규 생성)
- 세션 종료 조건: 별도 확정 필요 (시간제 또는 주문 생성 횟수 상한)
- 1시간 플레이 = 점심+저녁 세션 약 35회 기준
- 1시간 내 조리도구 추가 전체 구매 가능, 업그레이드는 장기 목표로 설계
- 에피소드 해금 업그레이드: 골드 고비용 + 해당 손님 드롭 아이템 필요

## 4. 인내심 타이머

동시 조리 기준 주문 1건 처리 소요 시간 약 10~15초 가정.
20초면 여유는 있되 방치는 안 되는 긴장감 유지.

| 손님 유형 | 점심 | 저녁 | 비고 |
|----------|------|------|------|
| 일반 손님 기본 | 20초 | 16초 | Day 1~3 기준 |
| 일반 손님 후반 | 16초 | 13초 | Day 5+ 적용 |
| 특별 손님 | 25초 | 20초 | 에피소드 연출 시간 확보 |

타이머 색상 전환 기준: 잔여 50% → 노랑, 잔여 25% → 빨강.

## 6. 업그레이드 비용

### 구분 기준

- **추가**: 조리도구/재료통 개수 증가 — 1시간 내 전체 구매 목표
- **업그레이드 (일반)**: 조리 속도 또는 골드 수급량 증가 — 수십 시간 장기 목표
- **업그레이드 (에피소드)**: 골드 고비용 + 해당 손님 드롭 아이템 필요 — 희귀 상위 업그레이드

### 추가 비용

| 항목 | 비용 | 최대 수량 |
|------|------|---------|
| 냄비 추가 | 3,000G / 개 | 기본 1 + 추가 3 = 최대 4개 |
| 냄비 재료박스 추가 | 2,000G / 개 | 기본 1 + 추가 2 = 최대 3개 |
| 프라이팬 추가 | 3,000G / 개 | 기본 1 + 추가 3 = 최대 4개 |
| 프라이팬 재료박스 추가 | 2,000G / 개 | 기본 1 + 추가 2 = 최대 3개 |
| 밥솥 추가 | 4,000G / 개 | 기본 1 + 추가 1 = 최대 2개 |
| 쌀통 추가 | 2,000G / 개 | 기본 1 + 추가 1 = 최대 2개 |
| 반찬박스 추가 | 2,000G / 개 | 기본 1 + 추가 1 = 최대 2개 |
| 쟁반 추가 | 5,000G / 개 | 기본 1 + 추가 n (상한 미확정) |

추가 총비용 합산 (최대치 기준): 약 44,000G

### 일반 업그레이드 비용

조리 속도 업그레이드는 각 조리기구별 적용.
골드 수급량 업그레이드는 각 재료박스/쌀통/반찬박스에 적용.

| 항목 | 효과 | 1단 | 2단 |
|------|------|-----|-----|
| 냄비 업그레이드 | 조리 속도 증가 | 20,000G | 50,000G |
| 냄비 재료박스 업그레이드 | 골드 수급량 증가 | 15,000G | 40,000G |
| 프라이팬 업그레이드 | 조리 속도 증가 | 20,000G | 50,000G |
| 프라이팬 재료박스 업그레이드 | 골드 수급량 증가 | 15,000G | 40,000G |
| 밥솥 업그레이드 | 조리 속도 증가 | 25,000G | 60,000G |
| 쌀통 업그레이드 | 골드 수급량 증가 | 15,000G | 40,000G |
| 반찬박스 업그레이드 | 골드 수급량 증가 | 15,000G | 40,000G |

### 에피소드 업그레이드 비용

해당 손님 에피소드 해금 이후 창고 메뉴에 항목 노출.
골드 + 해당 손님 드롭 아이템 동시 충족 시 구매 가능.

| 항목 | 효과 | 골드 비용 | 필요 드롭 아이템 | 해금 조건 |
|------|------|---------|--------------|---------|
| 냄비 특수 업그레이드 | 탄 음식 유예 시간 +5초 | 80,000G | 행운의 냄비 | 솜이 에피소드 완료 |
| 프라이팬 특수 업그레이드 | 동시 조리 슬롯 +1 | 100,000G | 대형 프라이팬 | 베어 에피소드 완료 |
| 밥솥 특수 업그레이드 | 1회 조리 시 5인분 생성 | 80,000G | (미확정) | 추후 결정 |
| 반찬통 특수 업그레이드 | 반찬 자동 세팅 | 80,000G | (미확정) | 추후 결정 |
