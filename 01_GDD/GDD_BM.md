## 1. 기본 방침

- 기본 게임 무료 (F2P)
- 핵심 게임 진행에 영향을 주는 모든 기능은 골드(플레이 재화)로만 구매 가능
- 젬(IAP 재화)은 외형 관련 아이템에만 연결

---

## 2. 수익 구조

| 항목 | 재화 | 비고 |
|------|------|------|
| 젬 패키지 IAP | 젬 | 직접 구매 |
| 캐릭터 스킨 번들 | 젬 | 직접 구매 또는 에피소드 드롭 |
| 조리도구 스킨 | 젬 | 기능 동일, 외형 변경만 |
| 특별 인테리어 | 젬 | 일반 인테리어는 골드로 구매 가능 |

---

## 3. BM 비연결 항목

- 조리도구 기능 업그레이드: 골드 전용 (BM 비연결)
- 특별 손님 해금 조건 조리기구: 골드 전용 (BM 비연결)
- 에피소드 드롭 아이템: 플레이 전용 보상 (BM과 별도)
- 젬→골드 환전 경로: 간접 허용 (젬→골드→조리기구, 직접 연결 아님)
