## 프로젝트 소개

**거북이식당**은 AI 오케스트레이션을 활용한 1인 힐링 아케이드 게임 개발 프로젝트입니다.
기획부터 코드, 아트, QA, 문서화까지 모든 과정에 AI를 적극 활용하여
초급 개발자 1인이 6개월 안에 완성 가능한지 검증하는 실험적 프로젝트입니다.

---

## 사용 기술 스택

| 분야 | 도구 |
|------|------|
| 게임 엔진 | Godot 4 (GDScript) |
| AI 코드 생성 | Claude Code, Cursor |
| AI 기획 | Claude.ai |
| AI 아트 | Midjourney, Adobe Firefly |
| 문서화 | Obsidian (Markdown) |
| 버전 관리 | Git |

---

## AI 오케스트레이션 실험

이 프로젝트의 핵심 주제는 **"AI를 어떻게 팀처럼 운영하는가"**입니다.

- 기획 에이전트, 코드 에이전트, 아트 에이전트, QA 에이전트, 문서 에이전트로 역할 분리
- 나(개발자)는 방향 결정 + 검토 + 승인만 담당
- 실제 코드·문서·에셋 생성은 AI가 담당
